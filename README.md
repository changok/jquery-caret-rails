# Jquery::Caret::Rails

## Installation

Add it to your `Gemfile`:
```
gem 'jquery-caret-rails', '~> 1.0.0'
```

## Usage

You can add jquery-caret-rails to your `application.js` file using a require statement like this:

```
//= require jquery-caret
```
